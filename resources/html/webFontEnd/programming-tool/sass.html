sass笔记

基于ruby环境下的预处理css样式

样式风格

嵌套输出方式 nested    内部用"&"符号承接

                a{
                    float:left;
                    &:hover{
                        color:blue;
                    }
                }

以下方式需在命令行中编译

展开输出方式 expanded

紧凑输出方式 compact

压缩输出方式 compressed

声明变量

以"$"美元符号声明变量

例：$width:100px;

注：scss有全局变量和局部变量之分。变量声明后设置"!global"为全局变量；设置"!defalut"为默认值

混合声明变量

宏的概念，使用"mixin"定义宏，使用"include"来调用

                //定义
                @mixin border-radius{
                    -webkit-border-radius:5px;
                    border-radius:5px;
                }
                //调用
                button{
                    @include border-radius;
                }
宏传递参数

参数不限定数量，参数为数组时，在定义参数后添加"..."

                //单一参数            
                @mixin opacity($opacity:50){
                    opacity:$opacity/100;
                    filter:alpha(opacity=$opacity);
                }            
                //调用
                button{
                    @include opacity(80);
                }

                //多组参数
继承

"extend"+类名/id名/选择器   可继承多项

注：混合器不会引起css样式层叠，而继承存在css样式层叠

占位符

"%"+类名/id名/选择器

如果占位符未使用，则销毁
