Ajax
class="pointer">load()方法加载数据
$(selector).load(URL,data,callback)
url:路径；data:查询键值对;callback:回调函数
加载JSON格式数据
$.getJSON(url,[data],[callback])
url:请求地址，data为请求时发送的数据，可选项callback:请求成功后的回调函数。
加载js文件并执行
$.getScript(url,[callback])
url:请求地址，可选项callback:请求成功后的回调函数。
GET,POST
$.get(URL,callback)
 $.post(URL,data,callback)

$(selector).serialize()
获取账户和密码，可直接用于ajax请求

$.ajax([settings])
既可以向服务器发生请求，又可以获取服务器返回的数据
