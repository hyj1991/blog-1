递归与非递归

计算机解析程序时，首先分配内存。调用JavaScript函数时，会为JavaScript函数创建一个栈及其作用域。当执行JavaScript函数，就是将函数推入栈的顶部。而递归就是将函数反复推入栈，重复分配N个内存直至函数结束。任何可以被递归定义的函数，都可以改写成非递归的迭代程序。

递归

经典函数递归

                function factorial(number){
                    if(number==1){
                        return number;
                    }else{
                        return number*factorial(number-1);
                    }
                }
                factorical(5);

函数执行过程

                5*factorial(4)
                5*4*factorial(3)
                5*4*3factorial(2)
                5*4*3*2factorial(1)
非递归 
                function factorial(num){
                    var result = 1;
                    for(var i=num;i>=1;i--){
                        result *= i;
                    }
                    return result;
                }
                factorial(5);
