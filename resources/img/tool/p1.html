sass笔记

基于ruby环境下的预处理css样式

样式风格

嵌套输出方式 nested    内部用"&"符号承接

                a{
                    float:left;
                    &:hover{
                        color:blue;
                    }
                }

以下方式需在命令行中编译

展开输出方式 expanded

紧凑输出方式 compact

压缩输出方式 compressed

声明变量

以"$"美元符号声明变量

例：$width:100px;

注：scss有全局变量和局部变量之分。变量声明后设置"!global"为全局变量；设置"!defalut"为默认值

混合声明变量

宏的概念，使用"mixin"定义宏，使用"include"来调用

                //定义
                @mixin border-radius{
                    -webkit-border-radius:5px;
                    border-radius:5px;
                }
                //调用
                button{
                    @include border-radius;
                }
宏传递参数

参数不限定数量，参数为数组时，在定义参数后添加"..."

                //单一参数            
                @mixin opacity($opacity:50){
                    opacity:$opacity/100;
                    filter:alpha(opacity=$opacity);
                }            
                //调用
                button{
                    @include opacity(80);
                }

                //多组参数
继承

"extend"+类名/id名/选择器   可继承多项

注：混合器不会引起css样式层叠，而继承存在css样式层叠

占位符

"%"+类名/id名/选择器

如果占位符未使用，则销毁

sassScript支持六种数据类型

数值：支持加减乘除及大小判断   除法font:#{$font-size}/#{$line-height};

文本：使用"#{x+y}动态设置值" （无论是否有引号

颜色：支持运算

布尔

空值

值列表：nth函数 访问某项;   join 连接多项;   append 添加项;   @each 遍历并添加样式

判断语句

if格式   @if   空格   语句   空格   {   语句   }   空格

连用格式

                @if 1+1=2 {
                    width:30px;
                }@else{
                    width:60px;
                }

三目运算   @if   (   condition,true   ,false)

for循环

@for   $i   from   through   包括end

@for   $i   to       through   不包括end

$i 变量      start 起始值      end 结束值 

                @for $i from 1 though 3 {
                    .item-#{$i} {
                        width: 2em*$i;
                    }
                }
                
                //解析为
                .item-1 {width:2em;}
                .item-2 {width:4em;}
                .item-3 {width:6em;}
each循环

@each   $var   in   list

循环一个列表并取出对应的值

                @each $fruits in apple,orange,banana{
                    .#{fruits}-icon{
                        background-image:url("/路径/#{fruits}.png");
                    }
                }
                
                //解析为
                .apple-icon{background-image:apple.png;}
                .orange-icon{background-image:orange.png;}
                .banana-icon{background-image:banana.png;}
